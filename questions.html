All Questions

To retrieve a list of the 30 most recently active questions from the API use: 
  Stacked::Question.all
If you wish to gather all the questions with their respective body, specify so: 
  Stacked::Question.all(:body => true)
This method also supports pagination through the page and pagesize options: 
  Stacked::Question.all(:pagesize => 10, :page => 2)
This method is also aliased as Stacked::Question.active
